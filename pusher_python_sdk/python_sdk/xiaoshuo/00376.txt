所谓触发式定时魂导炮，指的是在发动攻击时，炮弹命中对手或者被阻挡后，会出现吸收周围一切能量并且进行引爆的能力。最可怕的是，这个引爆的时间是可控的。

向自己一名五级魂导师使用手抛式魂导器，这已经不能用班门弄斧来形容了。

与魂导炮弹爆炸时的巨大声响相比，它的爆炸给人的感觉是那么的微不足道，就像是一个西瓜被摔破了似的，甚至并不能让人有多么注意。

有毒？
