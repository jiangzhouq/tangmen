正文 第一百二十章 唐门绝学（下）
两个念头几乎在同一时间出现在米迦心中，他万万没有想到会遭遇这种情况，他甚至不知道贝贝使用的是什么手段。//&nbsp;高速更新//
但是，就在这一刻，一道光芒从天而降，硬是将他的身体笼罩在内。就在那枚触发式定时魂导炮炮弹爆炸的一瞬间，挡住了爆炸的光芒。
米迦面前已经变成了一片橘红色，他的目光瞬间就呆滞了，头皮一阵发麻，背后更是冷汗津津。他完全可以想象，如果这一道光芒来得晚一瞬，根本没来得及反应的他，此时已经被那橘红色的光芒彻底吞噬。他也将步自己伙伴们的后尘，永远地离开这个世界。可以说，生死一瞬。
出手救下他的，自然是天煞斗罗黄津绪。
之前的变化已经令这位封号斗罗再不敢有半分大意，在他全部魂技中唯一一个防御技能早已运转掌中，一发现不对，第一时间就保护住了被攻击的米迦。
贝贝恋上流露出一丝惋惜之色，喃喃地道：“看来，还是哥能力强。他放出这么多个蛋，都不如哥这一个好用啊!”
橘红色光芒缓缓收敛，就在这时，天空中响起一声闷雷，天气变得更加阴沉了。
可这份阴沉也比不上日月战队众人此时的心情。
毫无疑问，天煞斗罗出手，就以为着米迦败了，最令他们无法理解的是，他们甚至不知道米迦是怎么败的。
贝贝使用的金属球究竟是什么？就连史莱克学院这边，大多数人脸上神色都是茫然的。
暗器，那绝对是我们唐门的暗器，霍雨浩双拳紧握，暮光灼灼地看着比赛台上傲然而立的贝贝。
是的，在沉寂了无数年之后，唐门绝学终于又出现了，只不过，现在又还有多少人认识这门唐门绝学呢？
子母追魂夺命胆，强大的唐门暗器，那瞬间出现的一片黑影，正是金属球，也就是母胆爆开之后飞出的无数牛毛钢针。这些钢针之上都带有毒性，毒物的出现，更主要的是为了掩盖它们的存在。
子母追魂夺命胆在唐门暗器之中是能排进前十位的，专破各种防御性护照类能力，在如此近距离被如此之多的牛毛针正面命中，哪怕是乌鸡魂导护罩也只能减缓它们的射速，而最终却依然被刺中。也正是这些牛毛针，直接将那枚触发炮弹引爆的。但却不是背背故意的了，只能说是巧合。
米迦的身体，就那么在比赛台中垩央缓缓软倒了。天煞斗罗刚刚有些得意的心情顿时变成了脸部肌肉抽搐，一个箭步就来到了米迦面前讲他扶了起来，迅速检查他的身体。
“毒？”天煞斗罗惊讶地扭头看向贝贝。
贝贝耸了耸肩膀，道：“不是致命的剧毒，请您放心，等今天的比赛全部结束后，我自然会为他解毒的。但显然不能是现在，对吧。”
后面还有二二三战法的对抗，现在为对手解毒，那不就意味着要在之后的比赛中多一个敌人吗？
黄津绪还能说什么？只得亲自将米迦送了下去。
史莱克学院，再胜。
目前为止，史莱克学院参加个人赛的七人之中只有一人被淘汰，而日月战队这边却已经是三人市区了战斗力。比赛进行到了这里，个人赛似乎已经没有了悬念。
治疗魂师对毒素也没什么办法，他们只能治疗伤势，更何况，唐门的剧毒又岂是谁都能够解除的？
身为预备战队队长，贝贝这一手极为高明。对方一人中毒，在后面的比赛中必定投鼠忌器，如果真的死拼，那么，在比赛结束后，不给他解毒又如何？
“日月皇家魂导师学院，第四名参赛队员，梦红尘，出场。”
梦红尘缓步走上比赛台，她的相貌虽然比不上江楠楠，但也依旧是极美的少女了。年龄也和江楠楠差不多。一头长发是少见的酒红色，比马小桃的发色要深，就像是醇美的葡萄美酒，闪烁着淡淡的光泽。但她的眼眸并非红色，而是蓝色，这一点和兄长笑红尘并不一样。
他已经是今天个人淘汰赛上第四名出场的日月战队队员了，包括她在内，日月战「绝世唐门手残手打」队就只剩下四人能够出战，这其中还包括了她那位已经身受重伤的兄长。
团战，日月战队是赢了，但所有的一切却全都在按照王言部署进行着，史莱克战队在战术上可以说取得了压倒性的胜利。之前比赛中所有战术层面的隐忍在这一刻都是值得的。
看看吧，个人赛中史莱克还有谁，除了贝贝之外，史莱克接下来即将出场的是控制系战魂王凌落宸，白虎战魂帝戴钥衡，邪火凤凰战魂帝马小桃。而日月战队还剩下了谁呢？
在梦红尘之后，他们就只有食物系五级魂导师萧夏风、队长马如龙以及受重创的笑红尘了。
一对一，同级别魂导师面对同级魂师本应该是占据优势的。可是，史莱克学院出品，又有哪个不是怪物呢？
米迦为什么输了？因为他根本没想到贝贝使用的战术手段，在他们的调查中，贝贝最强大的是战场上的武魂变异，蓝电霸王龙向黄金圣龙的变异。他一直在防备着，所以一直都没有使用出自己最强的攻击手段。他就在那一颗子母追魂夺命胆上，有些不明不白地输了，根本没有发挥出全力。
战场上瞬息万变不假，但又何尝不是因为史莱克战队每一位队员的变态呢？
两名魂宗登场，已经干掉了三名魂王，五级魂导师啊！那后面他们的魂王、魂帝更会强大到什么程度？
不说戴钥衡，单是马小桃在前面比赛中所展现粗的强横就足以在接下来的比赛中成为史莱克这边的中流砥柱。
个人淘汰赛，日月战队还能有机会？接下来的二二三战法，他们还能有机会？
决赛形势急转直下，胜利的天平似乎已经倾斜向了史莱克学院一方。这就是史莱克，有着万年底蕴的怪物学院！
梦红尘就是在这种情况下踏上了比赛场地，面对蓝电霸王龙战魂宗贝贝。
“比赛开始。”伴随着裁判天煞斗罗大喝一声，这场日月战队已经完全输不起的比赛，开始了！
贝贝低喝一声，脚步滑动，整个人已经用最快的速度冲向对手。有了刚才的一场胜利，他已经完成了自己的基本任务。他也知道，同样的获胜方式不可能再有第二次，但就算如此，哪怕是输，他也要尽可能地消耗梦红尘的实力，给接下来即将登场的凌落宸打下坚实的基础。
台下，王言的心情并没有因为先前的胜利而放松，双眼紧盯着刚刚登场的梦红尘，不知道为什么，他心中有些不安的感觉。
史莱克学院这边杀手锏频出，可对手呢？日月皇家魂导师学院如果真的那么容易对付，也不会在数千年来成为史莱克学院的对手了。
笑红尘的实力，就已经充分引起了王言的注意，如果当时不是江楠楠的突然爆发，以从未展现过的能力突然限制住对手，再加上戴钥衡的全力攻击，恐怕他会给史莱克战队带来更大的麻烦。他还不到十五岁、
那么，梦红尘呢？
同样作为日月战队的杀手锏，笑红尘的能力是控制金属，那么梦红尘的能力又是什么？这个大大的问号一直都出现在王言脑海之中未曾解答。
面对贝贝的前冲，梦红尘看上去十分平静，也没有像其他伙伴那样一上来就释放魂导器直接展开远程进攻。脚下微动，迈开小碎步，反而是向贝贝迎去。与此同时，她身上也随之出现了极其明显的变化。
原本酒红色的长发变了，变得一片雪白，而她那双原本是冰蓝色的眼眸却在刹那间变成了血红色，仿佛要滴出鲜血一般的血红色。
无论是谁，突然看到这样一双眼睛也必然要为之震撼，甚至是恐惧。
看着他，贝贝甚至有种当初面对那位邪魂师死神使者的感受，一种难以形容的阴冷瞬间而至。
也恰巧就这这时候，天空中再次响起一声炸雷，细密的雨丝，从天而降。
身体向前，梦红尘双掌抬起，能够看到，包括她面「绝世唐门手残手打」部、双掌在内，所有裸露在外的肌肤这一刻都变成了雪白色，洁白如玉，晶莹剔透。唯有那一双血眸看上去是那么的突兀、惊悚。
而在她的双掌掌心之中，隐隐有淡青色在微微闪烁着。
此时的梦红尘，死好不像是一位魂导师，反而更像是一名魂师。
无论是史莱克战队队员们还是饕餮斗罗玄老和王言，竟是没有人看出她的武魂是什么。
玄老双眼微眯，眼底闪烁着惊讶的光芒，虽然他暂时还看不出梦红尘的武魂是什么，但可以想见，她的武魂绝不是普通魂师那么弱，就算比不上笑红尘，恐怕也不会相差太多。
笑红尘的金属武魂和魂导师无疑更加契合，未来前途无量，而且小小年纪就已经是魂王级强者，未来成长起来，实力可想而知。
而眼前的梦红尘显然是走了另一条路，与她的兄长并不以上。
就在所有人震惊于梦红尘的武魂时，比赛台上的双方正在以高速接近着，距离很快就拉近到了三十米范围内。
率先发起攻击的额是梦红尘。她抬起的双手同时向外划出一个半圆，然后朝着贝贝的方向推了出去。
直到此刻，她身上才闪现出魂环的光彩，两黄、两紫、一黑，标准的最佳魂环配比。与此同时，她身后，缓缓浮现出一个光影。
那是一直洁白如玉的蟾蜍，一双眼睛正像她此时这样血红。但在笑红尘身上出现的武魂兽化却并未在梦红尘身上出现。
“这是……”王言骤然瞪大了眼睛，他自从进入史莱克学院之后就遍阅学院各种典籍中见到过。
只见梦红尘推出的双掌之中，一股直径一尺的白色气流直奔贝贝而去，他身上的第一魂环也随之亮起。
只所以说她释放出的是气流而不是光芒，那是因为这气流看上去速度并不快，白中隐隐泛青，就像是一团雾气似的。
但有些奇异的是，空中落下的雨丝很自然地融入到这气流之中，居然令它显现出了如同漩涡一般的样子。
面对未知的武魂，贝贝当然不会大意。他的性格要比徐三石沉稳的多，前冲的步伐骤然停顿，身上第三魂环雷霆之怒率先亮起，紧接着第四魂环也随之闪耀。一下来，他竟然就用处了自己最强的攻击手段，雷霆龙首。
看到梦红尘的样子，贝贝就知道，自己想要获胜恐怕并不容易，梦红尘随处都透露着神秘，因此，他给自己制定的目标很简单，尽一切可能消耗对手。
万千雷电伴随着雷霆之怒悍然绽放，映照着贝贝右手边神兽的龙鳞蓝紫色光芒大放，强悍的龙威将周围的雨丝全部真开。与此同时，一声嘹亮的龙吟声响起，巨大的雷霆龙首悍然而出，在雷霆之怒的增幅下，这一击决不逊色于魂王级别的强者，
因为此时天空中已是雷鸣滚滚，雷霆龙首更是威势大增，直径足足超过一米的巨大龙首破开雨丝，瞬间就撞在了那浓浓的白雾漩涡之上。
也就在这个时候，梦红尘却又做出了一个令人意想不到的选择。她没有继续向前，更没有要硬碰的意思，脚尖点地，身体闪电般弹起，竟然以惊人的速度倒飞而出。
雷霆龙首的冲入，将她排除的白雾绞得支离破碎，并且带着呼啸的雷鸣与龙吟之声向她追去。
梦红尘接连两个起落竟然就凭借着惊人的弹跳力回到了比赛台边缘，然后，她猛地横越而出。
同样能够锁定对手的雷霆龙首，竟然没有去追她的身体，而是在轰然巨响声中撞击在了比赛台的防护罩之上。
万千雷电轰然爆发，在护罩上形成了一个直径超过三米的巨大雷暴光球，足足持续了数秒才缓缓消失。
而落地的梦红尘却是一动不动，并没有再主动出击或者是做什么其他的事情，只是看着远处比赛台上的贝贝。
贝贝站在那里，他的目光有些呆滞，眼神之中，甚至还带着几分难以置信。眼神中的光彩，却已经以惊人的速度暗淡着。
“扑通。”贝贝的身体，居然就俺么直挺挺地向后倒去，摔在了比赛台之上。而哪怕是身体接触比赛台之后反弹而起时，他也没有在做出半分的动作。
这一刻，不只是观众们看得目瞪口地啊，史莱克战队这边，所有人也是同时起身，一个个脸上充满了不可思议的神色。
怎么可能？这怎么可能？分明是贝贝的雷霆龙首击溃了对方的攻击。可是为什么他在完全没有和对方接触的情况下却倒地不起？
这究竟是怎么回事？究竟发生了什么？
就连比赛台上的天煞斗罗黄津绪都有些呆滞了，因为他也不知道贝贝是为何而倒的。无论从哪个角度来看，他都没有倒下的理由。
可是，他就是倒下了，这也意味着这场比赛日月皇家魂导师学院获得了胜利。
天煞斗罗黄津绪用一股柔和的魂力托着贝贝的身体缓缓飞到史莱克战队众人面前，霍雨浩赶忙上前接住自己的大师兄。他发现，贝贝脸色苍白，就是这么一会儿的工夫，他的呼吸居然已经变得十分微弱了，整个人的生命气机似乎都已经被封死了。更可怕的是，贝贝的皮肤颜色正在变白，就像梦红尘身上的白，却并没有她那种晶莹的通透感。
梦红尘缓缓走「绝世唐门手残手打」到比赛台中垩央，冷冽的目光看向史莱克战队的待战区方向，右手抬起，伸出食指指向待战区中的众人，冷冷地道：“下一个。”
凌落宸缓缓挺直腰杆，史莱克战队这边，第三个上场的正式轮到了她。
突然间，王言和霍雨浩几乎同时叫出声来。
“我知道他的武魂是什么了？”
“我想起来了。”
前一句是霍雨浩的声音，后一句来自王言，霍雨浩的判断来自于脑海中突然惊醒的冰碧帝皇蝎，而王言的判断，则来自于他博览群书的知识。
两人对视一眼，异口同声地说道：“她的武魂是……”
