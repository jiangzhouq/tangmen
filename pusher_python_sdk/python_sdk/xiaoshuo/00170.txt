斗兽区。
作为整个史莱克学院专属区域最大的一个，也是史莱克学院最大的建筑，斗兽区的占地面积足有武魂系所有教学楼、宿舍楼再加上史莱克广场那么大。
在整个斗罗大陆上，能够以学院之力圈养魂兽的，那绝对是凤毛麟角，能够达到史莱克学院斗兽区这种规模的更是绝无仅有，哪怕是各大帝国皇室都不会做这种事。圈养魂兽可不是那么容易的，要保持魂兽的野性，同时不同魂兽还有着不同的饮食习惯，还要保证安全，再加上对魂兽的驱策，投入之大，是普通魂师学院连想都不敢想的。
史莱克城每年的税收全都归于史莱克学院，而这庞大的款项中的三分之一，都投入到了斗兽区之中。可想而知，这斗兽区需要的投入是何等高昂了。
刚入学的学员们一般都会觉得学费贵，但只要是去过斗兽区的学员就都不会有这种感觉了。和他们的学费相比，史莱克学院的投入之大，简直就是萤火与皓月。
除了来自于史莱克城的税收之外，史莱克学院的另一笔收入就来自于魂导系那边了，每年都会有魂导系制作的魂导器对外出售。但出售对象仅限于天魂、斗灵、星罗三大帝国。并不包括日月帝国，这也是为什么严少哲说钱多多有钱的缘故。
斗兽区主要分为两大部分，一部分是学员们无法看到的，就是专门圈养魂兽的地方，被称之为兽圈。另一部分就是斗兽场了。
兽圈除了在地面以上之外的部分，地面以下还有一部分，当初在建造这座兽圈的时候，除了工匠之外，还有超过一百位的魂导师配合，以各种强大的魂导器来限制圈养的魂兽。同时，这里还要有强大的魂师和魂导师镇守。据说，这里甚至圈养有万年级别的魂兽。至于十万年，那就是不可能的了，想要生擒活捉十万年魂兽，这个难度虽然大，但也不是完全不可能成功的。但是，十万年魂兽对于任何魂师来说，都是巨大的财富。不夸张地说，一只十万年魂兽能够令一名魂师一飞冲天。十万年魂环加上十万年魂骨的四个强大技能，何等强悍？
斗兽场则是专门提供给史莱克学院学员们进行实战训练以及考核之用的地方。此时，二年级的学员们，就在陆续进入斗兽场，准备接受他们的升级考核。
斗兽场呈椭圆形，长直径一百八十七米，短直径一百五十五米，最高处有五十七米左右。从外围看，整个建筑分为四层，底部三层为连拱式建筑，每个拱门两侧有石柱支撑，第四层有壁柱装饰，正对着四个半径处有四扇大拱门，是登上斗兽场内部看台回廊的入口。斗兽场内部的看台，由低到高分为四组，在特殊情况下，史莱克学院会进行特定的区域区分。整个工程的好大、壮观，甚至可以用辉煌、宏伟来形容。
这个巨大的斗兽场能够容纳的观众大约是一万人，因为更多的面积被用于斗兽场背部，就像斗魂区一样，这里既可以同时开始多场斗兽比拼，也可以只进行一场单独的斗兽。对于武魂系高年级学员来说，这个地方绝不陌生，修为高的学员愿意来到这里锻炼自己的实战技巧，而对于修为较低的学员来说，这里简直就是梦魇一般。
四年级以后，每周每个班级都会有至少一次的斗兽场课程，而每一次课程的成绩都将计入学员的期末考核成绩。
霍雨浩、王冬和萧萧一起走入这大斗兽场的时候，都有些目不暇接的感觉，无论他们出身如何，这都是他们见过的最为宏伟的建筑。
“雨浩。”天梦冰蚕的声音突然在霍雨浩脑海中响起，尽管他已经习惯了天梦哥这种神出鬼没的方式，但还是吓了一跳。
